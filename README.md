How to run:
- Navigate to the folder containing the makefile
- Run the "make" command
- Run "/.program" to execute
- Input the desired average arrival rate, average service time, base quantum, A parameter, and B parameter.
- Hit enter, and simulator will run and print results.

This is a discrete time event simulator, representing a Round Robin scheduler that adjusts the quantum for each service based on the given parameters, and a randomly selected priority for each process.
